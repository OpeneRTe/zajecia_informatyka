## Zadanie: Renderowanie listy miast za pomocą `map`

### Cel
Ćwiczenie wykorzystania metody `map` do generowania elementów JSX w React.

---

## Treść zadania
Masz tablicę:
```js
const cities = ["Tarnów", "Kraków", "Rzeszów", "Warszawa"];
```

Twoim zadaniem jest utworzenie komponentu React o nazwie **CityList**, który:
- wyświetla nagłówek `Lista miast`,
- renderuje listę uporządkowaną (`<ol>`),
- generuje każdy element listy (`<li>`) przy użyciu metody `map`,
- każdy element posiada klucz (`key`) równy jego indeksowi.

---

## Podpowiedź kodu
```jsx
export default function CityList() {
  const cities = ["Tarnów", "Kraków", "Rzeszów", "Warszawa"];

  return (
    <div className="container my-4">
      <h2>Lista miast</h2>
      <ol>
        {cities.map((city, index) => (
          <li key={index}>{city}</li>
        ))}
      </ol>
    </div>
  );
}
```
